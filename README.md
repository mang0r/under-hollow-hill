# Under Hollow Hills — Traduction française

Traduction amateur et non officielle de *Under Hollow Hills*, un jeu de rôle de **Meguey Baker** et **D. Vincent Baker** ([lumpley games](http://lumpley.com)).

Ce projet est un aide de jeu à destination de ma table. Il ne se substitue pas au jeu original — **[achetez-le ici](http://lumpley.com)**.

## Avertissement légal

*Under Hollow Hills* est la propriété intellectuelle de Meguey Baker et D. Vincent Baker (lumpley games). Cette traduction est **non officielle, amateur et à usage personnel de table**. Elle n'est ni affiliée ni approuvée par les auteurs.

Ce projet n'a pas vocation à se pérenniser sous cette forme. S'il devait évoluer au-delà d'un simple aide de jeu entre amis, une demande d'autorisation serait adressée aux Baker au préalable.

Le matériel source (PDF, texte anglais original) n'est pas inclus dans ce dépôt.
